# 闲鱼价差机会发现与履约辅助系统

这是一个本地运行的轻量后台，用来辅助规划无物流商品、记录闲鱼行情、生成发布草稿，并在有人下单后判断是否能找到低价货源。

给 Codex 或开发接手者看的完整项目上下文见：[CODEX_PROJECT.md](CODEX_PROJECT.md)，开发过程见：[DEVELOPMENT.md](DEVELOPMENT.md)。

## 运行

```bash
python3 app.py
```

打开：

```text
http://127.0.0.1:8765
```

## 第一版能力

- 基础操作台：页面只保留自动选品、发布队列、当前商品、咨询回复、接单履约。
- 定价建议：用中位数、去极值均价、低价区间计算建议售价。
- 发布草稿：生成标题、描述、价格和风险提示。
- 发布填表：复制填表脚本，在闲鱼发布编辑页自动填入标题、价格、描述。
- 发布助手：锁定下一条待发布商品，打开闲鱼页面并复制对应填表脚本。
- 订单履约：买家下单后，可粘贴最新低价货源并重新排序；找不到则生成缺货回复。
- 后台自动选品：从内置无需人力服务的数字权益候选池中自动选择商品、生成草稿并加入发布队列。
- 咨询回复：根据商品、库存、问题类型生成回复话术。

页面不提供手动添加商品、机会扫描、行情采样调试面板或商品管理列表；这些会干扰“后台自动规划商品”的主流程。

## 重要边界

本工具只做运营辅助，不绕过登录、验证码、风控，也不自动付款。采购付款和平台验证环节需要人工确认。

## 后台自动选品与发布队列

点击“自动选品入队”后，系统会：

- 从内置数字权益候选池里选择更适合测试、且不依赖人工服务交付的商品；
- 根据参考行情计算建议售价、采购上限和预估利润；
- 自动创建商品和行情样本；
- 自动保存发布草稿；
- 把商品加入发布队列。

发布队列里的“复制填表”会生成闲鱼发布页填表脚本。脚本只填标题、价格、描述，不点击最终发布按钮；类目、图片、规则提示和最终发布仍需要人工确认。

“开始发布队列”会优先继续 `active` 商品，否则取下一条 `ready` 商品，复制填表脚本并打开闲鱼页面。进入发布编辑页后运行脚本即可填表，发布后回后台标记“已发布”。

## 发布填表辅助

1. 在发布队列点击“开始发布队列”。
2. 系统会锁定当前商品，并复制对应填表脚本。
3. 进入闲鱼发布编辑页后运行脚本。
4. 人工检查类目、图片、规则提示和最终发布按钮。

填表脚本不会点击发布，也不会处理登录、验证码或平台风控。

## 订单履约增强

买家准备下单时，先按商品关键词重新搜索低价货源，把候选结果粘贴到“最新低价货源”。系统会：

- 先保存这些最新候选货源；
- 用成交价减去最低利润和风险缓冲得到采购上限；
- 按低价优先列出可采购候选；
- 如果没有候选低于采购上限，生成缺货/取消建议。
