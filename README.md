# 闲鱼价差机会发现与履约辅助系统

这是一个本地运行的轻量后台，用来辅助规划无物流商品、记录闲鱼行情、生成发布草稿，并在有人下单后判断是否能找到低价货源。

## 运行

```bash
python3 app.py
```

打开：

```text
http://127.0.0.1:8765
```

## 第一版能力

- 商品池：维护无物流商品、成本、最低利润、风险缓冲。
- 行情采样：录入从闲鱼搜索结果看到的标题、价格、备注。
- 浏览器采样：复制采样脚本，在闲鱼搜索页运行后把页面价格导入本地后台。
- 定价建议：用中位数、去极值均价、低价区间计算建议售价。
- 发布草稿：生成标题、描述、价格和风险提示。
- 发布填表：复制填表脚本，在闲鱼发布编辑页自动填入标题、价格、描述。
- 机会扫描：批量录入关键词和价格，自动生成值得测试的商品机会榜。
- 订单履约：买家下单后，筛选低价候选货源；找不到则生成缺货回复。
- 咨询回复：根据商品、库存、问题类型生成回复话术。

## 重要边界

本工具只做运营辅助，不绕过登录、验证码、风控，也不自动付款。采购付款和平台验证环节需要人工确认。

## 行情采集辅助

1. 在商品详情里点击“打开闲鱼搜索”。
2. 回到后台点击“复制采样脚本”。
3. 等闲鱼搜索结果加载完成后，在搜索页地址栏粘贴脚本并运行，或把脚本保存成浏览器书签后点击。
4. 回到后台查看导入的价格样本和定价建议。

采样脚本只读取当前页面可见文字，不处理登录、验证码或风控页面。

## 发布填表辅助

1. 在商品详情里确认建议售价和发布草稿。
2. 点击“复制填表脚本”。
3. 进入闲鱼发布编辑页后运行脚本。
4. 人工检查类目、图片、规则提示和最终发布按钮。

填表脚本不会点击发布，也不会处理登录、验证码或平台风控。

## 商品机会批量扫描

在“机会扫描”里按行输入：

```text
会员月卡 | 15元 16元 18元 20元
某权益周卡 | 8 9.5 11 12
```

系统会按每个关键词计算建议售价、采购上限、预估利润和是否值得测试。值得测试的机会可以点击“转商品”，进入商品池继续生成发布草稿和履约建议。
